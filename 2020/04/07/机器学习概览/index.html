机器学习概览

Posted on 2020-04-07 | |
Words count in article:

​ 现如今，在我们的日常生活中，每天你都可能在不知不觉中使用了各种各样的机器学习算法，机器学习算法已经融入到了我们的生活当中，并逐渐依赖于机器学习所带来的好处。比如 当你每一次使用像诸如谷歌 (Google)或必应 (Bing) 的搜索引擎时，它们运作得如此之好的原因之一便是由Google或微软实现的一种学习算法可以“学会”如何对网页进行排名，并对用户进行展示。当你使用邮箱的时候，程序会对收件箱进行垃圾邮件和有效邮件的分类，使你在使用邮箱的过程中免受垃圾邮件的困扰。

​ 自上世纪以来，人工智能经历了从符号主义到连接主义的漫长的发展。人工智能可以大致分为行为主义，符号主义，和连接主义三个学派。行为主义是基于控制论的，用于构建感知和控制的系统，如控制机器人行走，单脚站立等。 符号主义是基于算术逻辑的，基于表达式的，当求解具体问题时，先将其映射为相应的表达式，通过解出表达式的解，从而解决具体的问题。如果你在求解某个具体问题时，能用if，else if条件语句，和相应的数学表达式进行表示， 其实这就是符号主义的方法，比较典型的就是专家系统。现如今对专家系统的使用已经越来越少。符号主义使计算机具备理性的思维，但是人类在实际生活中，不仅仅有理性思维，还有很多的感性思维。而连接主义就是在模拟人的这种感性思维，其在仿造我们人脑内的神经元连接关系，从而引出了很多的新概念：神经元，全神经网络，前向传播，反向传播，学习率。。

​ 直到现在，也没有一个对机器学习的标准的定义，其中比较经典的对于机器学习的定义是Tom Mitchell给出的：

“A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.”
