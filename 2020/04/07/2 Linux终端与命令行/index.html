打开终端，此时我们没有输入任何字符的情况下，有一行不知名的字符串，其包含的内容很丰富。

开头的john表示当前用户的名字。在每个人的电脑上会有所不同。@则代表at，译为 在…当中。@前面表示用户名，后面表示主机名字。所以说，在我的电脑中，我的主机名字为john-VirtualBox，john@john-VirtualBox表示john-VirtualBox主机当中的john用户。

～表示虚拟机的home目录，$符号则表示当前用户为普通用户。可以通过命令sudo su进入root用户，在Linux中，root用户拥有至高无上的权利，传说中的“删库跑路”就是root干的行当，如下：

常用快捷键：

在终端中，我们应尽量避免使用鼠标，因此，常用的快捷键是必不可少的，也为我们的业务场景提供了实用的技能，下面列举一些常用的快捷键：

Ctrl + L：清理终端上的历史内容，也即清屏，保持界面的干净整洁。clear命令能实现相同效果。

Ctrl + D：给系统传递EOF，告诉程序是时候退出执行了。

Ctrl + A：光标跳到一行命令的开头。

Ctrl + E：光标跳到一行命令的结尾。

Ctrl + U：删除所有在光标左侧的命令字符。

Ctrl + K：删除所有在光标右侧的命令字符。
